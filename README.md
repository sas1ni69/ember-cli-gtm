# Ember-cli-gtm

Easily add Google Tag Manager to your app.

## Installation

```bash
# From within your ember-cli project
npm install ember-cli-gtm
```

## Usage

First you'll need to configure your GTM App ID:

```javascript
// config/environment.js

ENV['ember-cli-gtm'] = {
  appId: '[YOUR_APP_ID]'
};
```
